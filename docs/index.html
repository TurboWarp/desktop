TurboWarp Desktop

Install TurboWarp Desktop for Windows

  1. 1
    Download the installer.
  2. 2
    Run the .exe file.
  3. 3
    If a Windows SmartScreen alert appears, click "More info" then "Run anyways"

Install TurboWarp Desktop for macOS

  1. 1
    Download the installer.
  2. 2
    Open the .dmg file. Move TurboWarp Desktop into Applications.
  3. 3
    Our binaries are not notarized, so additional steps are required. Open Finder, navigate to Applications, right click TurboWarp Desktop and select "Open" and then "Open" again.

Install TurboWarp Desktop with AppImage

  1. 1
    Download app. (x86 64-bit)

    Download app. (ARMv7l)
    Alternative downloads for other architectures:
    Download the package for your system's architecture:
    • x86 64-bit
    • x86 32-bit
    • ARM 64-bit
    • ARMv7l
  2. 2
    Mark as executable. (chmod +x *.AppImage or a checkbox in right click > Properties > Permissions)
  3. 3
    Run the .AppImage

Install TurboWarp Desktop with Snap

TurboWarp is published in the Snap Store with the name turbowarp-desktop. Currently only x86 64-bit is supported. Certain Scratch features (microphone, camera, and bluetooth) may require manual connections.

  1. 1
    Install snapd.
  2. 2
    sudo snap install turbowarp-desktop

Bugs should be reported on GitHub or on Scratch.

Is TurboWarp Desktop faster than regular TurboWarp?

Usually not, but it may be more stable as your computer won't unload it to save memory.
