TurboWarp Desktop

Use TurboWarp, the Scratch mod with a compiler, dark mode, addons, and more, from an app on your desktop. It even works when you're offline.

TurboWarp is not affiliated with Scratch, the Scratch Team, or the Scratch Foundation.

Changes from v0.10.0 to v0.10.1

  • Project compression feature introduced in v0.10.0 has been temporarily removed because it might be corrupting projects.
  • The translate extension still doesn't work, but the necessary client-side changes to make it work again have been made. It will automatically start working when server-side logic is finished.

Changes from v0.9.0 to v0.10.0

  • New addon: Grab single block with Ctrl key (enabled by default)
  • New addon: Hide new variables
  • New addon: Extra key support
  • Windows: You can now zoom the window with Ctrl+Equals, Ctrl+Minus, Ctrl+Zero
  • Windows: You can now make a window fullscreen by pressing F11
  • Linux: You can now zoom the window with Ctrl+Equals, Ctrl+Minus, Ctrl+Zero
  • Linux: You can now make a window fullscreen by pressing F11
  • project.json of exported projects is up to 50% smaller letting you fit more scripts into projects before hitting any size limits
  • The translate extension has been partially disabled: https://github.com/TurboWarp/scratch-gui/issues/322. We hope to re-enable it in the next update.
